# Post Service

This microservice handles all post-related operations in the Instagram backend project.

## Endpoints

| Method | Route              | Description             |
|--------|--------------------|-------------------------|
| POST   | /api/posts         | Create a new post       |
| GET    | /api/posts         | Get all posts           |
| GET    | /api/posts/:id     | Get single post by ID   |

## Technologies
- TypeScript
- Express.js
- MongoDB
- Mongoose
- JWT Auth
